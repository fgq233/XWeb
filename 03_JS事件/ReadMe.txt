二、常见事件
1、onclick			鼠标单击事件
2、ondblclick		鼠标双击事件
3、onkeyup			按下并释放键盘上一个键时触发
4、onchange			文本内容/下拉菜单中选项--发生改变时
5、onfocus			获得焦点事件：文本框等标签获得鼠标焦点
6、onblur			失去焦点事件：文本框等标签失去鼠标焦点
7、onmouseover		鼠标悬停事件：即鼠标停在图片等上方
8、onmouseout		鼠标移除事件：鼠标离开图片等所在区域
9、onload			网页文档加载事件
10、onupload			关闭网页时
11、onsubmit			表单提交事件
12、onreset			重置表单事件
13、oninput			用户输入事件
